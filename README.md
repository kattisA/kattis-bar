# Kattis Bar App
A fun site with drink recipes

## Description

A fun project for learning more about:
- Tailwind CSS
- Next.js
- Deploying Next on GitHub pages
- GitHub actions

### Executing program

* Run  `npm run dev`
* Deploy [locally](http://localhost:3000)
* For linting use:
```
npm run build
```
